# 사난기

산업안전기사 / 산업안전산업기사 **필기 시험 대비** 웹 앱 (HTML / CSS / JS, 모바일 최적화)

## 실행 방법

`index.html`을 브라우저로 열면 바로 동작합니다. (서버 불필요)

## 학습 모드

| 모드 | 설명 |
|---|---|
| **연습** | 과목당 랜덤 1문제 출제 → 정답 체크 → 즉시 채점 표시 → 다음 문제 / 다른 과목 이동 |
| **실전대비** | 실제 시험시간 타이머(기사 3시간 / 산업기사 2시간 30분), 과목 이동, 남은 문항 표시, 종료 시 채점 후 성적표 |

- 문제 출제 순서와 객관식 보기 순서는 매번 랜덤으로 섞입니다.
- 합격 기준: 과목당 40점 이상 + 전 과목 평균 60점 이상 (성적표에 합격/불합격 표시)

## 데이터 현황

| 자격증 | 상태 |
|---|---|
| 산업안전기사 | ✅ 2016~2022 기출 20회차, **2,225문제** ([js/bank-gisa.js](js/bank-gisa.js), 해설 95% 포함) |
| 산업안전산업기사 | ⏳ 자료 대기 중 (샘플 문제로 동작) |

- 기출 PDF에서 자동 파싱했으며, 본문 정답 표시와 문서 말미 정답표를 교차 검증함 (불일치 0건).
- 그림·표·FT도 등 시각자료가 필요한 문제 51개는 제외.
- 회차 간 중복 문제는 최신 회차 기준으로 병합.

## 문제은행 데이터 입력 방법

문제 데이터 구조는 [js/data.js](js/data.js)에 있습니다.
각 과목의 `questions` 배열에 아래 형식으로 문제를 추가하면 됩니다.

```js
{
  q: "산업안전보건법상 안전보건관리책임자의 업무가 아닌 것은?",
  choices: ["보기 1", "보기 2", "보기 3", "보기 4"],
  answer: 2,                    // 정답 보기의 인덱스 (0부터 시작, 위 예시는 "보기 3"이 정답)
  explanation: "해설 내용",      // 선택 사항 — 없으면 생략 가능
},
```

- `answer`는 **원본 보기 순서 기준** 인덱스입니다. 출제 시 보기 순서가 랜덤으로 섞여도 자동으로 정답이 추적됩니다.
- 과목당 문제를 20개 이상 넣으면 실전대비에서 그중 20문제가 랜덤 추출됩니다.

### 실제 데이터 입력 후 할 일

`js/data.js` 상단의 샘플 모드를 끕니다.

```js
const SAMPLE_MODE = false; // true → false 로 변경
```

이후 샘플 문제 자동 생성 블록(파일 하단)은 삭제해도 됩니다.

## 파일 구조

```
사난기/
├── index.html      # 전체 화면 (홈 / 연습 / 실전 / 채점 / 성적표)
├── css/style.css   # 모바일 우선 스타일
└── js/
    ├── data.js     # ★ 문제은행 데이터 (여기에 문제 입력)
    └── app.js      # 앱 로직 (랜덤 출제, 타이머, 채점)
```
